## 1. 项目目标

本项目研究语言模型输入 embedding 矩阵 `E` 与输出 unembedding / lm_head 矩阵 `U` 的几何结构，重点问题是：

1. 原始 checkpoint 中，untied 模型的 `E` 和 `U` 有什么系统性差异？
2. tied 模型的共享矩阵在这些静态谱性质上更像 untied 的 `E` 还是 `U`？
3. Base→Instruct 过程中，`E` 与 `U` 的变化量是否低秩、是否能被仿射形式解释？
4. 全局仿射更新与纯 LoRA / W-rank 更新相比，哪一种更能解释 BI 变化？
5. GCorr / affine / SVD 这些不同角度是否支持同一组 E/U 几何规律？

当前最稳的三条实验线是：

| 实验线 | 目录 | 研究对象 | 核心结论 |
|--------|------|----------|----------|
| 主线 GCorr / affine 结果源 | `cross_model_geometry/` | Task1-6，BI、系列内、跨系列、MoE、affine | 提供所有主结果表；BI-clean 几何高度保持，full-vocab affine R² 很高 |
| 原始模型静态 SVD / E-U 几何 | `analysis_eu_geometry/`、`analysis_1/` | 94 模型 checkpoint 的 E/U 行范数、mean、谱 | untied U 的 raw rank1 主轴效应显著强于 E；tied clean 更像 untied U |
| BI 变化量 SVD / affine / LoRA | `bi_analysis/`、`__tep/` | BI-clean 30 的 `ΔE`、`ΔU`、`A-I`、affine component、W-rank budget | untied U 的变化量比 E 更 affine-aligned；tied 行为也更接近 U |

## 3. 实验线 A：原始模型静态 SVD / E-U 几何

### 3.1 问题

在不考虑 Base→Instruct 变化的情况下，直接看 checkpoint 里的 `E` 与 `U`：

- untied 模型的 `U` 是否比 `E` 更有主轴 / mean direction 结构？
- tied 模型的共享矩阵更像 untied `E` 还是 untied `U`？
- 这种差异是否只是“U 的奇异谱整体更低秩”？

### 3.2 数据与方法

主要目录：

- `analysis_eu_geometry/`
- `analysis_1/`

输入：

- `extracts/<model>.safetensors`
- `extracts/<model>.info.json`
- `configs/models.yaml`

核心分析：

- 行范数统计：row norm mean/var/min/max。
- mean 强度：`‖μ‖ / mean(row norm)`。
- 静态谱：对 `M^T M` 做 economy SVD / eigendecomposition。
- raw 与 centered 谱对照：centered 后剥离 mean 主导的 rank1。
- 指标包括 rank1/rank5/rank10 energy、effective rank、participation ratio / d、top singular vector 与 mean direction 的对齐度。

### 3.3 主要结果

在排除 DeepSeek-V4 Flash/Pro 后的 **48 个 comparable untied 模型**中：

- **48/48：U 的 raw rank1 能量占比高于 E**。
- U 的 `‖μ‖ / mean(row norm)` 通常高于 E。
- U 的 top singular vector 与 mean direction 高度对齐。
- tied clean 34 的共享矩阵在这种 raw rank1 / mean 主轴结构上更接近 untied U，而不是 untied E。

但这不是“U 整体奇异谱更低秩”的结论。更精确的表述是：

> U 的 raw 谱集中主要体现在 rank1，且主要由 mean vector 强度与 top singular direction 的对齐驱动；rank2+ 并不表现为 U 全面更集中。effective rank、PR/d、centered spectrum 和 top-k 增量分析都支持这个区分。

也就是说，当前静态 SVD 结论不是“U 密度更高 / 谱整体更尖”，而是：

> untied U 有更强的 mean-dominated principal axis effect；tied 模型共享矩阵在这一点上更像 U。

## 4. 实验线 B：BI 变化量 SVD / affine / LoRA

### 4.1 问题

对同一模型的 Base→Instruct 变化，研究：

- `ΔE = E_instruct - E_base` 与 `ΔU = U_instruct - U_base` 的谱结构是否不同？
- `ΔU` 是否比 `ΔE` 更能被仿射形式解释？
- 最优仿射矩阵 `A` 相对单位阵的偏移 `A-I` 是否比直接变化量更低维、更集中？
- affine update 与纯 W-rank / LoRA 形式相比，哪个解释更多变化？

### 4.2 数据与方法

主要目录：

- `bi_analysis/`
- `__tep/`
- 上游结果来自 `cross_model_geometry/results/task6_base_instruct_full_vocab/`

主口径：

- **BI-clean 30 对**：17 tied + 13 untied。
- untied pair 分别分析 E 与 U。
- tied pair 只有共享矩阵，记作 E 侧一行，但解释时要与 untied U 对照。

核心对象：

```text
X = Base matrix
Y = Instruct matrix
ΔW = Y - X
Y ≈ X A + b
ΔW ≈ X(A-I) + 1 b^T
Z = [X, 1]
```

主要分析层：

1. **Full-vocab affine fit**：完整词表 id 对齐，拟合 `Y ≈ X A + b`，看 `R²`。
2. **`ΔW` 与 `A-I` 的 SVD**：比较直接变化量和仿射矩阵偏移的谱集中度。
3. **有效 affine 子空间**：对 `Z=[X,1]` 的有效左奇异子空间 `Q_K` 做投影，计算 `R²_aff,K`、`A5_K`、`A10_K` 等。
4. **Centered 分解**：`D=Y_c-X_c`，`P=X_c(A-I)`，`R=D-P`，看 `P/D`。
5. **Affine vs W-rank / LoRA budget**：比较 affine 形式和纯 W-rank 更新在同等 rank budget 下的解释能力。

### 4.3 主要结果

#### 4.3.1 BI-clean 的 full-vocab affine R² 很高

BI-clean 30 对中，当前 Task6 表给出的中位数约为：

```text
E_R2 median ≈ 0.9974
U_R2 median ≈ 0.9973
```

因此第一层结论是：

> 对非异常 Base-Instruct pair，E 和 U 的整体空间之间都存在很强的全局仿射关系。

这不是 U 独有现象。U 的特殊性出现在“变化量中有多少 affine-aligned component”这一层。

#### 4.3.2 `A-I` 比直接变化量更集中

BI-clean 30 对中：

```text
E_delta energy@5%h median ≈ 0.3518
A_delta energy@5%h median ≈ 0.5435
```

对应结论：

> Base-Instruct 的 embedding 差异本身偏高秩，但最优全局仿射变换相对单位阵的偏移 `A-I` 更低维、更集中。

这条结论主要来自 Task6 full-vocab affine / SVD。

#### 4.3.3 untied U 的变化量更 affine-aligned

在 BI-clean 的 **13 对 untied pair** 中，有效 affine 子空间分析的中位数为：

| 指标 | untied E | untied U |
|------|---------:|---------:|
| `r5` | 0.019 | 0.222 |
| `A5_K` | 0.048 | 0.829 |
| `R²_aff,K` | 0.036 | 0.330 |
| `R²_aff,K^(5)` | 0.001 | 0.133 |

解读：

- E 侧前几个主变化方向几乎不对齐有效 affine 子空间，接近随机。
- U 侧前 5/10 个主变化方向显著落在有效 affine 子空间里。
- U 侧全谱约三分之一变化能量可由有效 affine 子空间解释。
- tied 17 对的共享矩阵行为与 untied U 同档，不像 untied E。

更短的主结论：

> untied BI 中，`ΔU` 比 `ΔE` 更集中、更 affine-aligned；tied 更新在这个意义上也更像 U。

#### 4.3.4 affine 形式相对纯 W-rank / LoRA 更有优势，尤其在 U 侧

BI-clean 30 的 affine / W-rank budget 摘要：

| 指标 | E | U |
|------|--:|--:|
| `P/D` median | 0.1080 | 0.3098 |
| untied `P/D` median | 0.0491 | 0.3188 |
| affine wins @ W-rank 1 | 16/30 | 27/30 |
| affine wins @ W-rank 8 | 15/30 | 22/30 |

结论：

> U/lm_head 更新中的全局 affine component 更强；在相同 W-rank budget 下，affine 形式在 U 侧更常胜过纯 LoRA / W-rank 近似。
