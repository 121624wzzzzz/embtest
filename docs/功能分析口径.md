## 1. `cross_model_geometry/` — 主线实验包

### 1.1 定位

当前仓库的**核心分析代码与结果**所在。包含 Python 包 `cross_model_geometry`（位于 `src/`）、CLI 入口（`scripts/`）、实验结果（`results/`）。

**研究问题**：不同 LLM 的 Embedding (E) 与 Unembedding/LM Head (U) 矩阵之间是否存在稳定的几何关系？

### 1.2 分析口径

| 维度 | 口径 |
|------|------|
| 模型范围 | `configs/models.yaml` 注册的 **94 个模型** |
| BI pair | `configs/base_instruct_pairs.yaml` 定义的 **35 对** Base→Instruct |
| tied 判断 | **`actual_tied`**：`np.allclose(E, U, rtol=1e-5, atol=1e-5)`，优先于 config 声明的 `tie_word_embeddings` |
| Task1-4 | bootstrap n=100，每次采样 20000 tokens、5M pairs |
| Task5 | 子采样最多 24000 行，最小公共 token 数 5000 |
| Task6 | **全词表** id 对齐 (0..vocab_size−1)，不采样不排除 |

### 1.3 核心算法

| 模块 | 文件 | 算法 | 关键细节 |
|------|------|------|----------|
| GCorr | `metrics.py` | 流式 Pearson 相关 | 采样 m 个 token pair → 分别算 cos/euc/euc² → Pearson($d_X$, $d_Y$)；euc² 用极化恒等式避免显式差分张量 |
| GCorr bootstrap | `metrics.py` | 双 CUDA stream 并行 E+U | 显存 ≥ 2×32GB 时 E/U 两路并行，seed 分离 |
| 仿射拟合 (Task5) | `task5_affine.py` | 中心化 `torch.linalg.lstsq` | `Y_c ≈ X_c·A` → `b = μ_Y − μ_X·A`；多 GPU 线程池 |
| 仿射拟合 (Task6) | `full_vocab_affine.py` | **流式中心化 normal equations** | 三遍流式：① 累加 Σx, Σy, Σy² → μ, SS_tot；② 累加 G=X_cᵀX_c, C=X_cᵀY_c → A=G⁻¹C；③ 累加残差 SS_res → R² |
| A 诊断 | `full_vocab_affine.py` | Frobenius 范数比 | `‖A−I‖/‖I‖`, `tr(A)/(‖A‖·‖I‖)`, `‖offdiag‖/‖A‖`, `‖AᵀA−I‖/‖I‖` |
| SVD 能量 | `full_vocab_affine.py` | d×d Gram eigendecomposition | `G = MᵀM` → `eigvalsh(G)` → 降序能量 → rank/eff_rank/energy@k/energy@%h |
| Token 对齐 | `alignment.py` | id 对齐 / string 交集 | 同 vocab → id 对齐；异 vocab → token string 交集；排除 special tokens |
| 数据加载 | `data.py` | safetensors 流式读取 | 支持全量加载和按 row_ids 切片加载（用于 smoke validation） |

**GCorr 三种 metric 定义**：
- **cos**: `dot(xᵢ, xⱼ) / (‖xᵢ‖·‖xⱼ‖)` — 方向相似性
- **euc**: `‖xᵢ − xⱼ‖` — 绝对欧氏距离
- **euc²**: `‖xᵢ‖² + ‖xⱼ‖² − 2·dot(xᵢ, xⱼ)` — 平方欧氏距离（极化恒等式，避免显式差分）

**SVD 能量指标定义**（用于 Task6，在 hidden-dim 侧，即 d×d）：
- `rank_{p}`：`min k s.t. Σᵢ₌₁ᵏ sᵢ² / Σⱼ sⱼ² ≥ p`
- `effective_rank`：`exp(−Σ pᵢ ln pᵢ)`，其中 `pᵢ = sᵢ² / Σ sⱼ²`
- `energy_at_{k}`：前 k 个奇异值的累计能量占比
- `energy_at_{x}pct_h`：前 `⌈x%·d⌉` 个奇异值的累计能量占比（跨 hidden dim 可比）

**Task6 的四组 SVD 谱**：
| CSV 前缀 | 矩阵 | 含义 |
|----------|------|------|
| `E_delta_*` | `E_instruct − E_base` | E 侧原始变化谱 |
| `U_delta_*` | `U_instruct − U_base` | U 侧原始变化谱（tied 时复用 E_delta） |
| `A_delta_*` | `A_E − I` | E 仿射变换相对单位阵的偏移谱 |
| `U_A_delta_*` | `A_U − I` | U 仿射变换相对单位阵的偏移谱（tied 时复用 A_delta） |

## 2. `analysis_eu_geometry/` — 静态 E/U checkpoint 几何审计

### 2.1 定位

**独立子项目**，对全库 94 模型的 checkpoint 存盘 E/U 权重做三层自下而上的描述性几何审计。**不运行 GCorr，不做 affine fit。**

回答的问题：checkpoint 里的 E/U 行范数、mean vector、谱结构是否存在异常？这些静态几何特征如何解释主线实验中的特殊现象？

### 2.2 分析口径

| 维度 | 口径 |
|------|------|
| 模型范围 | **全库 94 模型**（与 `configs/models.yaml` 一致）；推荐干净基线 **clean-82**（排除全部 12 个错误案例） |
| 矩阵 | 每个模型 E 和 U；tied 时 Layer 2/3 只算 E，Layer 1 仍算 E+U |
| 总行数 | Layer 1: **188 行**；Layer 2/3: **144 行**（44 tied × 1 + 50 untied × 2）；合并时 tied 模型 U 行沿用 E 的 μ/谱指标） |
| tied 判断 | 使用 `info.json` 的 `tie_word_embeddings`（config 声明值），**非** `actual_tied`（`np.allclose` 实测值）。与 `cross_model_geometry`/`bi_analysis` 的口径不同 |
| 子口径 | static-all (94)、**clean-82** (34 tied + 48 untied) = static-tied-clean (34) ∪ static-untied-comparable (48)、static-untied-all (50)、static-tied-all (44) |
| BI 子集 | 35 对 BI 的 70 个模型：行范数 + pair delta；clean-82 下为 30 对（BI-clean） |

### 2.3 核心算法

**三层分析**，自下而上：

**Layer 1 — 行范数统计** (`row_norms.py`)：
```
对矩阵 M ∈ ℝ^{n×d}：
  row_norms[i] = ‖M[i,:]‖₂
  → mean, var, min, max（全部 n 行统计）
```
对 BI pair 额外计算 delta_pct = `(Instruct_mean − Base_mean) / Base_mean × 100`

**Layer 2 — μ-ratio** (`mu_ratio.py`)：
```
μ = (1/n) Σᵢ M[i,:]                     # mean vector
‖μ‖ = L2 norm of mean vector
mean_row_norm = (1/n) Σᵢ ‖M[i,:]‖
μ-ratio = ‖μ‖ / mean_row_norm           # mean 相对行范数的强度
```
支持 CPU 多进程并行（`ProcessPoolExecutor`）。

**Layer 3 — SVD 全谱** (`spectral.py`)：
```
G = (MᵀM) / n                            # d×d Gram（归一化）
eigvalsh(G) → 升序奇异值平方 s₁² ≤ ... ≤ s_d²
总能量 = Σ s_i²
p_i = s_i² / 总能量

核心输出:
  rank1_energy_frac = p_d                  # 最大奇异值能量占比
  participation_ratio = (Σ s_i²)² / Σ s_i⁴  = 1 / Σ p_i²
  effective_rank = exp(−Σ p_i ln p_i)
  PR/d = PR / d                           # 归一化各向同性度
  sigma_ratio = s_d / s₁                  # 条件数
  cos(v₁, μ̂)                             # top-1 右奇异向量与 mean 方向的对齐度

去中心化版本: M_c = M − μ → 同流程再算一遍（centered 指标）
  目的: 剥离 mean 主导的 rank1 后看残余谱结构
```

**合并层** (`features.py`)：
```
按 (model, matrix) 合并 Layer 1/2/3 → all_models_eu_features.csv (188行)
tied 模型的 E 指标沿用到 U 行
```

### 3.4 核心算法

#### 3.4.1 有效子空间分析 (`evaluate_affine_effective_subspace.py`)

**问题**：$\Delta W = Y − X$ 有多少能量落在设计矩阵 $Z = [X, \mathbf{1}_n]$ 的**低维有效子空间**里？

```
输入: X (Base), Y (Instruct), 均为 n×d
ΔW = Y − X
Z = [X, 1_n]  ∈ ℝ^{n×(d+1)}           # 设计矩阵

# SVD of Z (样本维左奇异向量)
U_z, S_z, _ = SVD(Z)
K = min k s.t. Σ_{t≤k} σ_t²(Z) / Σ σ_t²(Z) ≥ η    # η=0.90/0.95/0.99
Q_K = U_z[:, :K]                                    # 有效 affine 子空间

# SVD of ΔW (样本维左奇异向量)
U_dw, S_dw, _ = SVD(ΔW)

# 全谱 R²
R²_aff = ‖P_Z ΔW‖² / ‖ΔW‖²             # 全 col(Z) 投影
R²_aff,K = ‖Q_K Q_Kᵀ ΔW‖² / ‖ΔW‖²     # 有效子空间投影

# 第 i 方向的对齐度
a_i(K) = ‖Q_Kᵀ u_i‖²                   # u_i = U_dw[:, i]

# Rank-L 加权的对齐度
A_{L,K} = Σ_{i=1}^L σ_i²(ΔW) · a_i(K) / Σ_{i=1}^L σ_i²(ΔW)

# 分解链
r_L = Σ_{i=1}^L σ_i²(ΔW) / ‖ΔW‖²      # 前 L 方向能量占比
R²_aff,K^(L) = r_L · A_{L,K}           # 前 L 方向对总 ΔW 的 affine 贡献

# 随机基线
baseline = K/n                           # 随机方向落入 K 维子空间的概率
A_{L,K} / (K/n) ≫ 1 ⇔ 对齐显著非随机

# bias 分量
a_bias = (u₁ · 1_n/√n)²               # rank-1 方向中 bias 型能量占比
```

**与 Task6 SVD 的关键区别**：此处的 SVD 在**样本维**（n×d → n 个左奇异向量），Task6 的 SVD 在 **hidden-dim 维**（d×d Gram → d 个特征值）。两者是互补视角。

#### 3.4.2 D/P/R 分解 (`evaluate_pred_delta_common_spectrum.py`)

```
输入: X (Base), Y (Instruct), 均为 n×d

# 中心化
X_c = X − μ_X,  Y_c = Y − μ_Y

# 流式累加（与 Task6 相同的 centered normal equations）
S = Σ x_cᵀ x_c                         # X_cᵀ X_c
C = Σ x_cᵀ y_c                         # X_cᵀ Y_c
A = S⁻¹ C                              # centered least-squares affine map

# 三元分解
D = Y_c − X_c                          # centered raw delta
P = X_c (A − I)                        # affine prediction
R = Y_c − X_c A                        # residual (orthogonal to P)

# 分别算 Gram → eigvalsh → cumulative energy
G_d = DᵀD, G_p = PᵀP, G_r = RᵀR
→ C_D(k), C_P(k), C_R(k)  累计能量曲线

# 关键指标
identity_R2 = 1 − ‖D‖² / ‖Y_c‖²       # Base 直接当 Instruct 的效果
P/D = ‖P‖² / ‖D‖²                      # centered 仿射可解释占比（核心指标）
full_affine_gain = P/D
full_affine_R2 = identity_R2 + (‖D‖²/‖Y_c‖²) × (P/D)
```

**与 Task6 的关系**：此脚本独立重做了 centered normal equations 仿射拟合（与 Task6 同算法路径），并在此基础上额外做了 D/P/R 三元 Gram 分解 + common-k 谱。identity_R2 和 full_affine_R2 需要 SS_tot（= ‖Y_c‖²），而 Task6 CSV 未保存该值，因此 identity_R2 必须从原始矩阵重算。Task6 CSV 的 `R2_E`/`R2_U` 即 full_affine_R2，与脚本算出的 full_affine_R2 在数值上应一致（同算法、同数据），但当前无跨文件一致性校验。

#### 3.4.3 Aff/LoRA 参预算 (`evaluate_w_rank_budget.py`)

```
W-form LoRA rank r_w:
  参数量 = h + r_w(n + h)              # 含 bias h

Aff/LoRA rank r_aff:
  参数量 = h + 2h · r_aff              # 含 bias h

同参预算下: r_aff = floor(r_w · (n+h) / (2h)) ≈ r_w · n/(2d)

分别对 D 做 rank-r 截断 SVD 近似，比较解释能量占比
→ aff/W 比值 > 1 表示 affine 同参预算下更好
```

典型值：r_w=1 时 r_aff ≈ 8–61（untied U 中位 15）

#### 3.4.4 Hybrid (`evaluate_hybrid_affine_w_budget.py`)

```
同参预算下: P 用 rank-a affine + R 用 rank-q W-form
与 pure W-form rank-t 比较解释量
→ 当前没有覆盖 BI-clean 30 对的完整结果，脚本仅保留为重算入口
```

旧的部分模型结果已删除，不进入现行数据、汇总或结论。后续只能在 GPU 空闲时使用 `--all-clean` 完整重算30对。

#### 3.4.5 五元分解 (`compute_affine_pred_delta_svd.py`)

将 raw delta 分解为三个正交部分，各自算 Gram → SVD 能量：

```
Δ_raw = Y − X                                       # raw delta
Δ_centered = Y_c − X_c                              # centered delta
P_affine = X_c (A − I)                              # affine main term
Residual = Y_c − X_c A                              # residual
Mean_shift = n · (μ_Y − μ_X)(μ_Y − μ_X)ᵀ           # mean shift

能量关系: ‖Δ_raw‖² = ‖Δ_centered‖² + ‖Mean_shift‖²
          ‖Δ_centered‖² = ‖P_affine‖² + ‖Residual‖²
```

与 `evaluate_pred_delta_common_spectrum.py` 的区别：
- `compute_affine_pred_delta_svd.py`：对五个 Gram 矩阵各自做完整 `svd_energy_from_gram()`，输出完整的 rank/eff_rank/energy@k/energy@%h 指标
- `evaluate_pred_delta_common_spectrum.py`：只对 D/P/R 三个矩阵做 common-k 累积能量曲线，额外计算 identity_R2 和 full_affine_R2

#### 3.4.6 Raw-scale 增益转换 (`augment_w_rank_budget_raw_scale.py`)

```
centered gain → raw-scale gain 的转换:
  gain_raw = mean_shift/raw + (D/raw) × gain_centered

其中 mean_shift/raw 和 D/raw 从 affine final summary 表读取
```

将 centered 口径下的 Aff/W LoRA budget 结果转换为 raw-scale 口径（计入 mean shift 的贡献）。

### 4.4 核心叙事（`notes/02`）

**主命题**：$\Delta W$ 的总变化中，有相当一部分可以被 $[X, \mathbf{1}_n]$ 诱导的有效 affine 子空间解释；这些可解释部分集中在 $\Delta W$ 的主变化方向上。

**关键公式链**：
```
R²_aff,K = Σ_i σ_i²(ΔW) · a_i(K) / Σ_i σ_i²(ΔW)
         = Σ_i (ΔW 第 i 方向能量) × (该方向对 Q_K 的对齐度)

分解: R²_aff,K^(L) = r_L · A_{L,K}
  r_L: 前 L 方向能量占比（谱集中度）
  A_{L,K}: 前 L 方向的加权 affine 对齐度

结论: U 的 r_L ↑ × A_{L,K} ↑ ⇒ R²_aff,K ≈ 33%（E ≈ 4%）
```

**关键结论表**（untied 13 对，η=95%，中位数）：
| 指标 | E | U | 解读 |
|------|:--:|:--:|------|
| A₅,K | 0.048 | **0.829** | U 前 5 主方向 ~83% 对齐 Q_K |
| A₅/(K/n) | 1.6× | **30.3×** | U 对齐 ~30 倍随机，E ≈ 随机 |
| R²_aff,K | 3.6% | **33.0%** | 全谱 affine 可解释占比 |
| P/D | 4.9% | **31.9%** | centered 线性可解释占比 |
| R/D | — | **13/13 U < E** | U 仿射残差占比系统性更低 |
| r₅ | 1.9% | **22.2%** | U 谱更集中 |
| aff/W @r1 | 0.34× | **2.18×** | U 侧 aff 同参预算显著优于 W |

1. **口径统一**：`__tep/affine/`、`bi_analysis/` 与全仓主叙事均使用 BI-clean 30
